# Hemocard

Landing page em React + Vite baseada no protótipo visual do projeto Hemocard.

## Como rodar

```bash
npm install
npm run dev
```

## Como gerar build

```bash
npm run build
```

## Identidade visual

- Fonte: Poppins
- Vermelho principal: `#BB1D1D`
- Branco: `#FFFFFF`
- Cinza-claro: `#EDEDED`
- Escuro de apoio: `#282828`
